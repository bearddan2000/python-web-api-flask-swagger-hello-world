# python-web-api-flask-swagger-hello-world

## Description
Creates a simple api for a flask project.
Document endpoints with swagger ui.

## Tech stack
- python
- flask

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`
- API http://localhost
- Swagger UI http://localhost/swagger

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
